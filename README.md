# Bootstrap 5 Starter Template 🎨
A template with some useful packages.

# Ingredients 🥣
- Node.js
- Popper.js
- Node-Sass
- Bootstrap
- Bootstrap-Icons
- Font-Awesome

## Build Setup 🛠️

```bash
# install packages
$ npm install

# Execute the "sass" script in order to work with Bootstrap
$ npm run sass
```
